# neovim-setting

# Abstract
This is nvim settings for making terminal to be used as ide for C, C++, Python, Rust.
It includes code formatting for the languages above.
The base configuration is configured by NvChad ver 2.5

# How to use
- Download neovim and NvChad.
- Follow the instruction on the official website of those two
  - https://nvchad.com/
  - https://neovim.io/
- Follow the settings in this repo's source code file
- Use MasonInstallAll to download the required plugins
- You can add plugins for other language and debuggers that this setting does not include
